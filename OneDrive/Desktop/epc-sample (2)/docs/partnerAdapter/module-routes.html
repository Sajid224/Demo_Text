Module: routes

This file contains the Partner Adapter's routes. Each route is an endpoint that will be used by a client to interact with the Partner Adapter.
  • Functions that start with handleUI are intended to be called by the Partner UI. The Partner UI will use AJAX to make Partner Adapter requests.
  • Functions that start with handleEPC are intended to be called by Encompass Partner Connect. The route will be configured as part of the Partner Product configuration.
