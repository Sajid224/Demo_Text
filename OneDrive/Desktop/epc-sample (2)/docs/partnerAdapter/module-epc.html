(inner) getOrigin(originId, PAT) → {Promise.<Origin>}

This function will contact Encompass Partner Connect to get the origin information. The origin information is intended to help the Partner UI render. It is important that the Partner UI does not include the loan data in the 'options' section.
Parameters:
Name Type Description
originId string The origin id that is generated in the Partner UI using the Partner JavaScript API
PAT string The partner access token (PAT) that is generated in the Partner UI using the Partner JavaScript API
